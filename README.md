## About

A Node.js command-line application that dynamically generates a professional README.md file from user input using Inquirer prompts. Answer a series of questions about your project and receive a polished, structured README with a table of contents, license badge, and contact section.

## Features

- Interactive CLI prompts for all README sections
- Auto-generated table of contents
- License badge selection
- Description, installation, usage, and contribution sections
- Contact info with GitHub and email links
- Output saved as a separate markdown file

## Tech Stack

| Layer | Technology |
|-------|------------|
| Runtime | Node.js |
| CLI | Inquirer 8 |
| Output | Markdown (.md) |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/coleyrockin/ProREADMEGen.git
cd ProREADMEGen

# Install dependencies
npm install

# Run the application
node index.js
```

The generated README will be saved as `Readme1.md` in the project root.
